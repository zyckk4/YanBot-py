# jieba_words.txt

本文件存储了一些敏感词，有些敏感词会被 jieba 错误的断句<br>
比如“你妈死了”一般来说断句为“你妈/死/了”，但是jieba会断句为“你/妈/死/了”<br>
本文件用于增强jieba分词的正确率，你同样可以加一些词
